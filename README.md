# FitClub - Sistema de Gestão de Treinos 

Este projeto é uma API desenvolvida para modernizar a gestão de treinos em academias. Ele nasceu da minha trajetória como estudante de **Análise e Desenvolvimento de Sistemas na PUC Minas** e reflete meu compromisso em criar soluções que facilitem a vida do usuário final.

##  Objetivo do Projeto
O FitClub foi criado para resolver uma dor real: a dificuldade de organizar e acompanhar a evolução de treinos de forma digital e intuitiva. Como profissional que valoriza o **Sucesso do Cliente**, foquei em construir uma estrutura robusta que garanta uma experiência fluida e sem erros para quem utiliza o sistema.

##  Tecnologias Utilizadas
* **Backend:** Java 17 com Spring Boot 3.2.5
* **Banco de Dados:** MongoDB (NoSQL) para alta escalabilidade.
* **Segurança:** Autenticação via JWT (JSON Web Token).
* **Documentação:** Swagger/OpenAPI (Focando na transparência para quem integra com o sistema).
* **Gestão:** Gradle.

##  Diferenciais de Negócio e CS
Além do código, este projeto foi pensado sob a ótica de **Customer Success**:
- **Interface Intuitiva:** Estrutura de dados pensada na facilidade de uso do aluno e do instrutor.
- **Confiabilidade:** Tratamento global de exceções para garantir que o usuário nunca fique sem resposta.
- **Documentação Clara:** Facilita o Onboarding de novos desenvolvedores ou parceiros que queiram consumir a API.

##  Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/wesley-hs/fitclub-api.git`
2. Certifique-se de ter o MongoDB ativo.
3. Execute: `./gradlew bootRun`
4. Acesse a documentação em: `http://localhost:8080/swagger-ui.html`
